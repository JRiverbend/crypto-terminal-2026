## Описание проекта (RU)

**Crypto Strategy Terminal** — это персональный дашборд для мониторинга структурных изменений на крипторынке в 2026 году. В отличие от обычных трекеров цен, этот инструмент фокусируется на **фундаментальном анализе**: балансе спроса и предложения, активности китов и эффективности инфраструктуры L2.

### Основные возможности:
* **Whale & Supply Analysis:** Отслеживание перемещения крупных капиталов и анализ «шока предложения» (Supply Shock).
* **Institutional Demand:** Мониторинг притоков в ETF и спроса со стороны крупных фондов.
* **L2 Efficiency Tracker:** Анализ прибыльности сетей второго уровня (Arbitrum, Base, Optimism).
* **The Graph (GRT) Utility:** Мониторинг реального использования протокола индексации данных.
* **Anti-Crisis Dashboard:** Быстрый взгляд на ключевые метрики для принятия взвешенных решений в условиях высокой волатильности.

### Технологический стек:
* **Language:** Python 3.8+
* **Framework:** Streamlit (Cloud-native UI)
* **Data Sources:** CoinGecko API, Mempool.space, Whale Alert (simulated/API).

### Установка и запуск (Local):
1. Установи зависимости: `pip install -r requirements.txt`
2. Запусти терминал: `streamlit run app.py`

## Project Description (EN)

**Crypto Strategy Terminal** is a personalized dashboard designed for monitoring structural shifts in the crypto market in 2026. Unlike generic price trackers, this tool focuses on **fundamental analysis**: supply/demand balance, whale movements, and L2 infrastructure efficiency.

### Key Features:
* **Whale & Supply Analysis:** Tracking large capital movements and analyzing "Supply Shock" dynamics.
* **Institutional Demand:** Monitoring ETF inflows and demand from major institutional players.
* **L2 Efficiency Tracker:** Profitability analysis for Layer 2 networks (Arbitrum, Base, Optimism).
* **The Graph (GRT) Utility:** Real-time monitoring of indexing protocol usage and fees.
* **Anti-Crisis Dashboard:** A high-level overview of critical metrics for informed decision-making during volatility.

### Tech Stack:
* **Language:** Python 3.8+
* **Framework:** Streamlit
* **Target Hardware:** Optimized for lightweight operation (MacBook Air 2017+).
